# Task 5.1: Media and Content Apps

This repository contains two Android applications demonstrating modern Android architecture, media playback, and data persistence.

## Project Overview

### 1. Sports News Feed App
- **Architecture**: Built using a Single Activity Architecture with multiple Fragments.
- **Key Features**: 
  - Horizontal RecyclerView for featured matches.
  - Vertical RecyclerView for latest sports news.
  - Real-time search and filter bar to filter news by category.
  - Bookmark system to save favorite stories locally.
- **Tech Stack**: Fragments, RecyclerView, Custom Adapters.

### 2. iStream - Personal Video Playlist App
- **Authentication**: Secure Login and Sign-up system using Room Database.
- **Data Isolation**: Ensures each user can only access their own private video playlist.
- **Media Playback**: Integration of YouTube videos via URL using the YouTube iFrame API.
- **Tech Stack**: Room Persistence Library, YouTube API, Intent.

## Key Technologies
- **Language**: Kotlin
- **Architecture**: Single Activity with Fragment Navigation
- **Database**: Room (SQLite)
- **UI Components**: Material Design, RecyclerView, TextWatcher
